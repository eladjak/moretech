🗺️ תוכנית פעולה — שלב אחר שלב

סדר עדיפויות: הפעולות מסודרות לפי סדר — כל שלב תלוי בקודם. עקוב מלמעלה למטה.

🔴 שלב 1: רכישת דומיין (יום 1)

רכוש edutech.co.il

זה הדומיין המרכזי של הפלטפורמה. בלעדיו אי אפשר להשיק.

אפשרות א' — Cloudflare (מומלץ):
1. היכנס ל-dash.cloudflare.com
2. תפריט שמאלי → "Register Domains"
3. חפש edutech.co.il
4. לחץ Purchase → הזן פרטי תשלום
5. מחיר: ~₪50/שנה

אפשרות ב' — Box.co.il (אם Cloudflare לא תומך .co.il):
1. היכנס ל-box.co.il
2. חפש edutech.co.il → רכוש
3. בהמשך נעביר ל-Cloudflare DNS
10 דקות

הגדר DNS ב-Cloudflare

אם רכשת ב-Cloudflare — ה-DNS כבר מוגדר. אם ב-Box — שנה nameservers.

אם רכשת ב-Box:
1. ב-Cloudflare: "Add Site" → הזן edutech.co.il
2. Cloudflare ייתן 2 nameservers
3. ב-Box: הגדרות דומיין → שנה nameservers לאלה של Cloudflare
4. חכה עד 24 שעות (בדרך כלל 1-2 שעות)
15 דקות + המתנה

🟡 שלב 2: הקמת Supabase (יום 1-2)

צור פרויקט Supabase חדש

Supabase מספק Auth + Database + Storage בחינם (עד 50K MAU).

1. היכנס ל-supabase.com/dashboard
2. לחץ "New Project"
3. שם: edutech-platform
4. סיסמת DB: שמור במקום בטוח
5. Region: eu-central-1 (הכי קרוב לישראל)
6. לחץ "Create new project" → חכה 2 דקות
5 דקות

העתק מפתחות לפרויקט

צריך URL + anon key כדי שהאפליקציה תתחבר ל-Supabase.

1. ב-Supabase Dashboard: Settings → API
2. העתק: Project URL + anon / public key
3. צור קובץ platform/.env.local:

NEXT_PUBLIC_SUPABASE_URL=https://xxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGci...
3 דקות

צור טבלת beta_signups

כדי לשמור הרשמות מטופס הבטא.

ב-Supabase: SQL Editor → הרץ:

CREATE TABLE beta_signups (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  full_name TEXT NOT NULL,
  email TEXT NOT NULL UNIQUE,
  subject TEXT,
  grade_level TEXT,
  school TEXT,
  created_at TIMESTAMPTZ DEFAULT now()
);

ALTER TABLE beta_signups ENABLE ROW LEVEL SECURITY;
CREATE POLICY "Allow inserts" ON beta_signups
  FOR INSERT WITH CHECK (true);
5 דקות

🟡 שלב 3: מפתחות AI API (יום 2)

הגדר מפתח Anthropic או Google

כדי שמחולל השיעורים יעבוד, צריך לפחות מפתח אחד.

אפשרות א' — Anthropic (Claude, מומלץ):
1. console.anthropic.com → API Keys → Create Key
2. הוסף ל-.env.local: ANTHROPIC_API_KEY=sk-ant-...

אפשרות ב' — Google (Gemini, חינם):
1. aistudio.google.com/apikey → Create API Key
2. הוסף ל-.env.local: GOOGLE_GENERATIVE_AI_API_KEY=AIza...
5 דקות

בדוק שהמחולל עובד מקומית

לפני שמפרסים — וודא שהכל עובד.

1. פתח טרמינל בתיקיית platform/
2. הרץ: npm run dev
3. פתח http://localhost:3000 → בדוק דף נחיתה
4. לחץ "נסו את מחולל השיעורים"
5. בחר מקצוע + כיתה + נושא → לחץ "צרו תכנית שיעור"
6. וודא שתכנית שיעור מופיעה ב-streaming בעברית
10 דקות

🟢 שלב 4: פריסה ב-Vercel (יום 2-3)

פרוס ב-Vercel

Vercel = הבית של Next.js. פריסה אוטומטית מ-GitHub.

1. vercel.com/dashboard → "Add New" → "Project"
2. חבר את הריפו eladjak/edutech
3. Root Directory: הגדר ל-platform
4. Environment Variables: הוסף את כל המפתחות:
  • NEXT_PUBLIC_SUPABASE_URL
  • NEXT_PUBLIC_SUPABASE_ANON_KEY
  • ANTHROPIC_API_KEY (או GOOGLE_GENERATIVE_AI_API_KEY)
5. לחץ "Deploy" → חכה ~1-2 דקות
10 דקות

חבר דומיין ל-Vercel

אחרי שהדומיין נרכש וה-DNS ב-Cloudflare.

1. ב-Vercel: Project → Settings → Domains
2. הזן: edutech.co.il → Add
3. Vercel ייתן הוראות DNS:
  • A record → 76.76.21.21
  • CNAME: www → cname.vercel-dns.com
4. ב-Cloudflare: DNS → Add Record → הוסף את שתי הרשומות
5. SSL אוטומטי
15 דקות
