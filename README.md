# MoreTech

MoreTech is a B2B EdTech company focused on training existing teachers to become technology and computer science educators. The project includes a marketing website, pitch deck, business plan, and outreach materials.

**Live Site:** [eladjak.github.io/moretech](https://eladjak.github.io/moretech/)

## Overview

Israel faces a growing shortage of technology teachers in schools. MoreTech addresses this by offering a comprehensive package: training programs, curriculum materials, mentoring, and a management dashboard -- empowering schools to build internal tech teaching capacity.

## Repository Contents

## Getting Started

### Run the Website Locally

```bash
git clone https://github.com/eladjak/moretech.git
cd moretech/website
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
