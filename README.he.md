# MoreTech - מורטק

MoreTech היא חברת EdTech בתחום B2B שמתמקדת בהכשרת מורים קיימים להפוך למורי טכנולוגיה ומדעי המחשב. הפרויקט כולל אתר שיווקי, מצגת משקיעים, תוכנית עסקית וחומרי פנייה.

**אתר חי:** [eladjak.github.io/moretech](https://eladjak.github.io/moretech/)

## סקירה

ישראל מתמודדת עם מחסור גובר במורי טכנולוגיה בבתי הספר. MoreTech מציעה חבילה מקיפה: תוכניות הכשרה, חומרי לימוד, ליווי מנטורי ודשבורד ניהול -- שמאפשרים לבתי ספר לבנות יכולת הוראת טכנולוגיה פנימית.

## תוכן המאגר

| תיקייה | תיאור |
|---------|-------|
| `website/` | אתר שיווקי (Next.js 16 + Tailwind CSS + Framer Motion) |
| `pitch-deck/` | מצגת משקיעים (PPTX + שקפי HTML + תסריט הצגה) |
| `business-plan/` | תוכנית עסקית מלאה עם תחזיות פיננסיות |
| `outreach/` | תבניות מיילים לפנייה לרשתות בתי ספר |
| `competitive-analysis.md` | ניתוח 13 מתחרים + SWOT |
| `RESEARCH.md` | מחקר שוק ונתונים |

## טכנולוגיות (אתר)

| טכנולוגיה | שימוש |

## התקנה

### הרצת האתר מקומית

```bash
git clone https://github.com/eladjak/moretech.git
cd moretech/website
bun install
bun dev
```

פתחו [http://localhost:3000](http://localhost:3000) בדפדפן.
